# IND250 Final Project

## Course Overview
IND250, Introduction to Basic Computer Integrated Manufacturing, is a course taught at Piedmont Virginia Community College that encompasses all aspects of computer integrated manufacturing systems: Robotics, Conveyor Control, and Machining Center Integration Quality Control, Statistical Quality Control, and Computer Integrated Manufacturing (CIM) software. I dual-enrolled this course my Fall 2023 semester.

## Project Overview
For my final project, I designed a unique part, inspired by my dog, using SOLIDWORKS CAD software. I then wrote G-code for my part and milled it using our labratory's CNC (Computer Numerical Control) Mill.

## Project Contraints
1. Four unique milled features
2. At least one drilled hole
3. 3" x 3" aluminum stock
4. 1/2" and 1/4" end mill
